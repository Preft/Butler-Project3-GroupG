Evalute, design, and deploy a machine learning model

Brent's model Using the ICO data, Brent used 10 categories (disappearance(consumption), domestic-consumption, exports, stocks, imports, indicator prices, inventories, prices paid to growers, retail prices, and production) to create a model. This model would attempt to predict changes to the retail price of coffee beans when there were changes to exports, prices paid to growers, production, and consumption. In the end, it was difficult to find a strong correlation between these data features. The lack of a strong correlation lead to low testing scores. Therefore, Brent looked for an additional data feature to add to the model, which turned out to be weather. Using data from World Bank temperature and precipitation were added as features to the model. The weather data only included the years 1990 to 2012 which meant Unfortunately, the weather data had a low positive correlation to predict it from 2013-2018 to fill in missing data created model with that data to predict average retail price they aligned 70-90 percent correct using the positive correlations to predict the future years
